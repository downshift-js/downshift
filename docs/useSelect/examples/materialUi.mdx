# useSelect

## Readme

The `useSelect` hook provides functionality and accessibility to a dropdown that should act as a HTML `<select>`, as described
by the corresponding [ARIA docs](https://www.w3.org/TR/wai-aria-practices/examples/listbox/listbox-collapsible.html).

User should be allowed to perform a selection after opening the dropdown's menu. Only one item may be selected at a time.
The widget should be built using a text label (optional but recommended), a button that toggles the menu, the menu itself
and the collection of items that belong in the menu.

## MaterialUI

A dropdown created with MaterialUI components such as `<FormLabel>`, `<List>`, `<ListItem>` and `<Button>`. Also child components
that are needed to display the `<ListItem>`.

https://material-ui.com/

<Playground style={{height: '150px'}}>
  {() => {
    const classes = useStyles()
    const DropdownSelectMaterialUI = () => {
      const {
        isOpen,
        selectedItem,
        getToggleButtonProps,
        getLabelProps,
        getMenuProps,
        highlightedIndex,
        getItemProps,
      } = useSelect({
        items,
        itemToString: item => item.primary,
      })
      return (
        <>
          <FormLabel {...getLabelProps()}>My Cloud:</FormLabel>
          <Button
            variant="contained"
            color="secondary"
            className={classes.button}
            {...getToggleButtonProps()}
          >
            {selectedItem ? selectedItem.primary : 'Folders'}
            <ExpandMoreIcon className={classes.rightIcon} />
          </Button>
          <List className={classes.root} {...getMenuProps()}>
            {isOpen && items.map((option, index) => {
              const itemProps = getItemProps({item: option, index, refKey: 'itemRef'})
              const {itemRef} = itemProps
              delete itemProps.itemRef
              return (
                <ListItem 
                  key={`${option.primary}-${index}`} 
                  className={index === highlightedIndex ? classes.highlighted : undefined} 
                  ref={itemRef} 
                  {...itemProps}
                >
                  <ListItemAvatar>
                    <Avatar>
                      {React.createElement(option.icon)}
                    </Avatar>
                  </ListItemAvatar>
                  <ListItemText primary={option.primary} secondary={option.secondary} />
                </ListItem>
              )
            })}
          </List>
        </>
      )
    }
    return <DropdownSelectMaterialUI />
}}
